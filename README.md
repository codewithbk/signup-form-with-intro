## 😊 Challenges

The most difficult part of this project was creating the password validation system.

I wanted to show live feedback while the user was typing. Managing different password rules, updating the progress bar, and keeping the UI responsive took several attempts before I was happy with the result.

I also spent time improving the validation logic instead of only checking if the inputs were empty.
